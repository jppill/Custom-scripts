# Markdown
#### Titre
> # H1
```
#H1
```
> ## H2
```
##H2
```
> ### H3
```
###H3
```
---
> **bold text** __bold text__
```
**bold text** __bold text__
```

> *italicized text* _italicized text_
```
*italicized text* _italicized text_
```

> ***bold and italic text*** ___bold and italic text___
```
***bold and italic text*** ___bold and italic text___
```
